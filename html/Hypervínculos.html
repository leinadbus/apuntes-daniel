Redacción de enlaces

Usaremos el comando <a mas el elemento href=".."

    <p><a href="https://firefox.com/">Descargar Firefox</a></p>

Resultado:

Descargar Firefox

Si queremos que el enlace se abra en una pestaña nueva deberemos incluir el elemento target="blank"

    <p><a href="https://firefox.com/" target="blank">Descargar Firefox</a></p>

Menú de navegación

Convinando el comando de lista desordenada visto anteriormente y el comando de enlace, podemos crear un menú de navegación entre archivos .html.

Para ello debemos crear varios documentos .html y enlazarlos con el comando <a añadiendo el nombre del documento .html en el elemento href="..".

Todos los documentos que estas leyendo estan enlazados entre sí como un menú de navegación, por tanto, son un ejemplo del mismo.

Enlace a correo electrónico

El mismo sistema de comando y elemento anteriormente visto pero añadiendo el elemento mailto: despues de href.

    <p><a href="mailto:correo@example.com">Enlace a correo electrónico</></p>

Resultado:

Enlace a correo electrónico

Especidifcar detalles de envío

Igual que podemos enlazar a un correo electrónico, podemos pre definir elementos como el cuerpo del mensaje, el asunto, si enviar copia o si enviar con copia oculta.

Estos son los elemtos <subject> para el asunto del mail, <body> para el cuerpo, <cc> para hacer "con copia a" y <bcc> para hacer "con copia oculta a". 

    <p><a href="mailto:email@example.com?cc=leinadbus3@gmail.com&bcc=leinadbus3@gmail.com&subject=El%20asunto%20del%20email&body=El
                %20Cuerpo%20del%20correo"> Enviar un correo electrónico cc, bcc, asunto y cuerpo 
               <0a><p/>

Resultado>

Enviar un correo electrónico cc, bcc, asunto y cuerpo

Añadir información de asistencia

Esta información nos facilita conocer mas información sobre a donde se dirige el enlace, u otros usos que puedas darle

Esto se utiliza con el comando <a con el elemento title=".."

Enlace a directorio y subcarpetas

A veces los documentos a los que queramos acceder no estarán en la misma carpeta donde se encuentra el documento .html actual. Para acceder a ellos debemos usar dos caracteres que le dirán a la máquina que acceda a otras carpetas o que salga de las mismas. Estos elementos son "/" y "../"

Dentro del comando <a href=".." deberemos decirle a la máquina donde se encuentra el documento que queremos que use, ya sea una imagen, un archivo .html, un archivo .css o un archivo de cualquier índole que queramos utrilizar. Para ello imaginemos que queremos enlazar otro documento .html en una lista desordenada, pero dicho documento se encuentra en una carpeta llamada CarpetadeJuan y dentro de esta carpeta el archivo al que queremos enlazar se llama archivo_de_juan.html.

    Bajar en directorios:

    
                 <ul>
                 <p><a href="CarpetadeJuan/archivo_de_juan.html"> Example</a></p>
                 </ul>

Imaginemos ahora que nos encontramos programando el archivo_de_juan.html, y queremos introducir un enlace dentro del archivo a un archivo que se encuentra fuera de la CarpetadeJuan, el archivo al que queremos enlazar se llama, por ejemplo, página_principal.html. Para ello debemos decirle a la máquina que salga de la CarpetadeJuan y busque en esa carpeta superior el archivo 

    
                <ul>
                <p><a href="../página_principal.html"> Example</a></p>
                </ul>

Enlazar a una descarga

Podemos decirle al navegardor, no solo que se dirija al documento que le digamos, sino que descargue el archivo que pueda encontrarse dentro de dicho documento, esto lo haremos usando el elemento download 

    
                <p><a href="https://download.mozilla.org/?product=firefox-latest-ssl&os=win64&lang=es-MX"
                download="firefox-latest-64bit-installer.exe">Descarga la última versión de Firefox para Windows (64 bits) (Español, es-MX)
                 </a> Cuidado que la descarga de verdad xD</p>

Resultado:

Descarga la última versión de Firefox para Windows (64 bits) (Español, es-MX) Cuidado que la descarga de verdad xD
