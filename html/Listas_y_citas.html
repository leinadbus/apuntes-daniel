Lista desordenada

Utilizamos los comando <ul> para crear el espacio de trabajo de la lista y el elemento <li> para determinar entradas de la misma. 

    
                  <ul>
                  <p>Lista de la compra:</p>
                    <li>Patatas</li>
                    <li>Coca-Cola</li>
                    <li>Nata montada</li>
                  </ul>

Resultado:

    Lista de la compra:

  • Patatas
  • Coca-Cola
  • Nata montada

Lista ordenada

Exactamente el mismo sistema anterior pero en vez de usar <ul> usaremos <ol>, lo que creará una lista enumerada automáticamente. 

    
                  <ol>
                  <p>Lista de la compra:</p>
                  <li>Patatas</li>
                  <li>Coca-Cola</li>
                  <li>Nata montada</li>
                  </ol>

Resultado:

    Lista de la compra:

  1. Patatas
  2. Coca-Cola
  3. Nata montada

Lista anidada o mixta

Podemos mezclar los dos elementos anteiores como más nos guste si fuera necesario

Cita en bloque independiente (Blockquote o salto de página con margen)

Si una sección de contenido a nivel de bloque se cita en otro lugar, debes envolverla dentro de un elemento <blockquote>.

El elemento <blockquote> indica que el texto se limita a una información que existe en otro lado

Para incluir una URL que apunte a la fuente de la información utilizamos el atributo cite

    <blockquote cite="https://www.hazunaweb.com/curso-de-html/blockquote-span-nobr-center/#:~:text=El%20blockquote%20se%20usa%20para,observar%20el%20efecto%20del%20blockquote."> <p>El elemento HTML <blockquote> indica que el texto al que delimita es una cita ampliada</p> </blockquote>

Resultado:

El elemento HTML <blockquote> indica que el texto al que delimita es una cita ampliada

Tablas

A continuación vamos a aprender como hacer tablas en HTML. Para comenzar deberemos enmarcar la tabla en el elemento <table>, esto limita la información contenida en la tabla. Para entender como vamos a realizar las tablas debemos saber que las tablas en HTML se realizan por filas olvidando las columnas, ya que, separando la información que incluimos en las filas creamos las columnas, es decir, para hacer una tabla de tres columnas y tres filas, escribiremos tres filas con tres separaciones que serán las que creen las columnas.

La primera fila que escribimos en una tabla siempre es la fila que da título a las columnas y a los que vamos a incluir en ellas, esta fila se escribe entre el comando <tr> y cada apartado de la fila a partir del comando <th>. Después de la fila de títulos las demás filas también se escriben entre el comando <tr> y cada casilla de ta tabla entre el comando <td>. Vamos a verlo con un ejemplo de una tabla de 4x3 (4 filas con 3 columnas): 

                
              <table>

                <tr>
              
                  <th>Hoy</th>
              
                  <th>Mañana</th>
              
                  <th>Viernes</th>
               
                </tr>
              
                <tr>
              
                  <td>Soleado</td>
              
                  <td>Mayormente soleado</td>
              
                  <td>Parcialmente nublado</td>
              
                </tr>
              
                <tr>
              
                  <td>19°C</td>
              
                  <td>17°C</td>
              
                  <td>12°C</td>
              
                </tr>
              
                <tr>
              
                  <td>E 13 km/h</td>
              
                  <td>E 11 km/h</td>
              
                  <td>S 16 km/h</td>
              
                </tr>
              
              </table>

Resultado:

Hoy Mañana Viernes
Soleado Mayormente soleado Parcialmente nublado
19°C 17°C 12°C
E 13 km/h E 11 km/h S 16 km/h
