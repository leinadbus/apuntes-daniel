Énfasis de texto

De la palabra "Emphasis" para dar enfasis a las palabras usamos el comando <em>

    Me <em>alegro</em> de que no llegues <em>tarde</em>.

Resultado:
Me alegro de que no llegues tarde.

Negrita en texto

Se utiliza el comando <strong> para enmarcar palabras o frases importantes. El resultado visible es como la negrita, pero se uso principalmente para que los lectores de pantalla para ciegos entiendan qué palabras o frases son importantes y resalten su importancia en el contexto. 

    Este líquido es altamente <strong>tóxico</strong>.

Resultado:
Este líquido es altamente tóxico.

Itálica, Bold y lineado

Estos casos no suelen ser usados gracias a la integración de CSS, sin embargo, viene bien conocerlos para realizar ejercicios o para casos puntuales.
Los comandos con los que enmarcaremos las palabras o frases son <i> para el formato Italic, el comando <b> para el formato bold y el comando <u> para el formato underline. 

    El <i>spanglish</i> es <u>comunmente usado</u> por los <b>paletos</b>.

Resultado:
El spanglish es comunmente usado por los paletos.

Superíndice y subíndice

Se usan para marcar cosas como fechas o elementos químicos. Estos comando son <sub> para los subíndices y <sup> para el superíndice. 

                       
    Nací en el 25<sup>th</sup> de mayo de 2001.
    
                   
    La fórmula química de la cafeína es C<sub>8</sub>H<sub>10</sub>N<,sub>4</sub>O<sub>2</sub>.
    
                   
    Si x<sup>2</sup> es 9, x debe ser igual a 3 o -3.

Resultado:

Nací el 25th de mayo de 2001.

La fórmula química de la cafeína es C8H10N4O2.

If x2 es 9, x debe ser igual 3 o -3.

Representación de código informático

Estos son los elementos disponibles para marcar código:

  • <code>: Para marcar fragmentos genéricos de código informático.
  • <pre>: Para respetar los espacios en blanco (en general, en los bloques de código) — si utilizas la sangría o diversos espacios en blanco consecutivos dentro de un texto, los navegadores los ignorarán y no se mostrarán en la página. Sin embargo, si delimitas el texto con las etiquetas <pre%gt; y %ñt;/pre> los espacios en blanco se representarán de forma idéntica a como se ven en tu editor de texto.
  • <var>: Para marcar específicamente nombres de variables.
  • <kdb>: Para marcar entradas de teclado (y de otro tipo) en el ordenador.
  • <samp>: Para marcar la salida de un programa de ordenador.

    Abreviaturas

    Un elemento que se usa para envolver una breviaturao acrónimo, que propiorciona un texto emergente que define la misma, usando el comando <abbr seguido de un atributo title="..." 

      <p>Usamos <abbr title="Lenguaje de marcado de hipertexto">HTML</abbr> para estructurar
                   nuestros documentos web.</p>

    Resultado:

    Usamos HTML para estructurar nuestros documentos web.

    Marcar la información de contacto

    Utilizando el elemento <adress> se etiqueta la información de contacto.

      <adress><p> Daniel Rodríguez, Soto del Real, Madrid, España.</></adress>

    Resultado;

    Daniel Rodríguez, Soto del Real, Madrid, España.

    Marcar fecha y hora

    Usamos el elemento <time datetime="..">, es usado para que la máquina pueda leerlo,

    Esto es útil si, por ejemplo, la web está siendo leida para invidentes o si queremos recoger todas las fechas en un calendario automáticamente.

      <time datetime="2021-04-06"> 6 de Abril de 2021</time>

    Resultado;

    6 de Abril de 2021

    <br> y <hr>

    <br>: Se utiliza para dar un salto de línea.

      <p>En un lugar de la mancha <br> de cuyo nombre no quiero acordarme.

    Resultado:

    En un lugar de la mancha
    de cuyo nombre no quiero acordarme.

    <hr>: Se utiliza para crear una línea horizontal que corte la página

      <p>En un lugar de la mancha <br> de cuyo nombre no quiero acordarme.
      
          <hr>
          Un ¿hidalgo don caballero?

      Resultado:

      En un lugar de la mancha
      de cuyo nombre no quiero acordarme.

      Un ¿hidalgo caballero?
